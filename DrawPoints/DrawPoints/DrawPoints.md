# DrawPoints
1. GetPixel() -> 화면 (x, y)지점의 색을 얻는다.
2. SetPixel() -> 화면 (x, y)지점에 특정 색상의 점을 찍고 원래 점의 색을 리턴한다.
3. SetPixelV() -> SetPixel() 함수와 출력은 같지만 원래 점의 색을 리턴하지 않으므로 속도가 좀 더 빠르다.
4. 윈도우의 크기를 변경하면 WM_PAINT 메시지가 발생하고 OnPaint() 함수가 호출되어 다시 점을 찍게되는데 점을 찍는 속도가 느리기 때문에 화면에 그림이 그려지는 과정이 보인다.
5. 점을 하나씩 찍어 비트맵 이미지를 출력하는 것은 너무 느리다.
6. 시스템 메모리에 점을 모두 찍은 다음 한꺼번에 메모리로 전송하는 것이 빠르다.
7. 블록 단위 데이터를 고속으로 전송하는 기능을 Blt(Block Transfer)라고 한다.
8. Blt 기능은 메모리 DC를 통해 수행한다.
9. 메모리 DC는 CDC::CreateCompatibleDC() 함수를 통해 생성한다.
