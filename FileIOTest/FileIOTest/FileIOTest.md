# CFile 클래스
파일 관련 MFC의 최상위 클래스이다.
# 파일 열기
CFile::Open() 함수의 첫 번째 인자는 파일명이고 두 번째 인자는 접근과 공유 모드를 나타내는 플래그 값이다. ex) CFile::modeRead(읽기 전용 모드로 연다.), CFile::modeReadWrite(읽기 및 쓰기 모드로 연다.), CFile::modeCreate(파일을 무조건 생성한다.)
# 파일 닫기
객체 소멸 시 자동으로 파일을 닫기 때문에 파일을 닫는 함수를 명시적으로 호출할 필요는 없다. 명시적으로 닫을 땐 "'CFile 객체명'.close()"를 해준다.
# 파일 읽기와 쓰기
CFile::Read() 와 CFile::Write() 함수를 이용한다. 읽고 쓸 버퍼의 주소와 전송할 바이트 수를 인수로 넣어준다.
# CStdioFile
CStdioFile 클래스는 텍스트 파일을 줄 단위로 읽고 쓸 수 있는 ReadString()과 WriteString() 함수를 제공한다.
# 도큐먼트 뷰 구조
1. 도큐먼트 -> 데이터를 저장하거나 읽고 데이터의 변경 사항이 생기면 뷰를 갱신한다.
2. 뷰 -> 데이터를 화면에 보여주고 사용자와 상호작용한다.
3. 하나의 도큐먼트와 작업하는 프로그램을 SDI 응용 프로그램이라고 하고 여러개의 도큐먼트와 작업하는 프로그램을 MDI 응용 프로그램이라고 한다.
